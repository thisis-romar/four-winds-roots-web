# Domain Shortlist — `.ca` Candidates

Availability verified 2026-05-28 via DNS `Resolve-DnsName` against authoritative `.ca` servers (NXDOMAIN = AVAILABLE, NOERROR = TAKEN). Final purchase requires a 30-second confirmation at <https://lookup.cira.ca/> before committing.

## Ranking weights

- Brand exact match (40 %)
- Already pointed-at by existing `schema.json` / `BaseLayout.astro` canonical / og:url (15 %) — registering means zero code changes
- Short + memorable + speakable for voice search and word-of-mouth referrals (15 %)
- Inclusion of `pharmacy` keyword for local-search intent (10 %)
- No hyphens, no number stylizations (10 %)
- Defensibility — alias / typo / hyper-local capture (10 %)

## Recommended action

### Minimum (1 domain, ~CAD $25 first year)

Register **`fourwindsrootspharmacy.ca`** at any CIRA-accredited registrar (Hover, Tucows, Cira-managed.ca, namespro.ca). Attach as the Netlify custom domain. **Zero schema/code changes required** — the placeholders already point here.

### Recommended bundle (3 domains, ~CAD $70 first year)

1. `fourwindsrootspharmacy.ca` — primary, attached to Netlify
2. `fourwindsroots.ca` — short alias, 301 → primary (for voice + business cards + email signature)
3. `rootsfourwinds.ca` — typo/alternate-order defense, 301 → primary

### Defensive bundle (add 2 more, ~CAD $50 more)
